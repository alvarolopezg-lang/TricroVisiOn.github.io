## 🗂️ Estructura del sitio

```
index.html        ← Página principal (única página del sitio)
README.md         ← Este archivo
```

El sitio es una sola página HTML con las siguientes secciones:

- **Inicio** — Portada con título y datos del curso
- **Proyecto** — Descripción del problema y la solución
- **Equipo** — Tarjetas de los 5 integrantes
- **Semanas** — Bitácora con pestañas por semana
  - Semana 1 — *(próximamente)*
  - Semana 2 — *(próximamente)*
  - Semana 3 — Modelado 3D individual (5 bloques con enlaces a Tinkercad)
- **Evidencia** — Galería de imágenes del proceso

## 🚀 Cómo publicar en GitHub Pages

1. Sube `index.html` y `README.md` a este repositorio
2. Ve a **Settings → Pages**
3. En *Branch*, selecciona `main` y carpeta `/root`
4. Guarda — en 1-2 minutos el sitio estará en vivo

---

## ✏️ Cómo editar el contenido

Abre `index.html` con cualquier editor de texto (Notepad, VS Code, etc.) y busca los textos entre corchetes `[...]` para reemplazarlos:

| Placeholder | Qué reemplazar |
|---|---|
| `[Nombre Completo]` | Nombre real del integrante |
| `[Nombre del Docente]` | Nombre del profesor del curso |
| `[2024-I / 2024-II]` | Ciclo académico actual |
| `[Sección del grupo]` | Sección asignada |
| `[Placeholder — Describe...]` | Texto real de cada sección |
| `href="#"` en botones Tinkercad | URL pública del modelo en Tinkercad |

Para **agregar una foto** de integrante, reemplaza el bloque `<div class="member-photo">A1</div>` por una etiqueta `<img>` con la ruta de la foto.
